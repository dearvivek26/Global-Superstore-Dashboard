# Global-Superstore-Dashboard



This Power BI dashboard provides a comprehensive analysis of the Global Superstore dataset. It visualizes key performance metrics such as Total Sales (7.47M) and Total Profit (843.31K) across multiple dimensions. The dashboard includes insights on sales trends by year and month, quarterly performance, and category-wise analysis (Furniture, Office Supplies, Technology). Filters for market, region, and segment allow dynamic data exploration. It helps identify top-performing years, profitable regions, and seasonal sales patterns, supporting strategic business decisions.
